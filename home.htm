The missing solutions of your MIT-missing-semester class

Edition 2020

Completed in 2024 by Simon Bouchard

Using windows subsystem for linux




This page was created to add solutions to the 2020 MIT missing semester course exercises. It contains the solutions to the exercisesi of sessions 1 through 9 as well as some suggested exercises and their solutions for session 10 (there are no real exercises for sessions 10 and 11).

Feel free to suggest changes and improvements to this page and to the solutions of the exercises via a pull request to the page's github repository (you will learn how through the course of this class).

I completed the exercises using ubuntu 22.04 on Wsl windows subsystem for linux.
My dotfiles that I developped whilst completing this class are availible here.

You can contribute to this page here.
